What is the Document object Module(DOM)?

When a web page is loaded, the browser creates a Document Object Model of the page.With the object model, JavaScript gets all the power it needs to create dynamic HTML :

  • JavaScript can change all the HTML elements , attribute in the page
  • JavaScript can change all the HTML attributes in the page
  • JavaScript can change all the CSS styles in the page
  • JavaScript can create new HTML events in the page

How do you select an element from the DOM?

To select an element from the DOM (Document Object Model) in JavaScript, there are several methods you can use depending on how you want to target the element. Below are the most common methods:

  • document.getElementById()
  • document.querySelector()
  • document.querySelectorAll()
  • document.getElementsByClassName()
  • document.getElementsByTagName()
Use getElementById when you know the element has a unique ID.
Use querySelector for more complex or flexible CSS-style selections.
Use querySelectorAll or getElementsByClassName when selecting multiple elements.

What is event Deligation in the context of the DOM and why is it usefull

Event delegation is a technique in JavaScript where instead of attaching event listeners to specific child elements, you attach a single event listener to a parent element. The event then "bubbles up" from the child to the parent, allowing you to handle events for multiple child elements with just one listener.
This is particularly useful in scenarios where you have many child elements, or if child elements are dynamically added to the DOM, and you want to handle events on all of them efficiently.
